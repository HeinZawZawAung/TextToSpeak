  • facebook access token with key "fbtoken".
  • Method:Post
  • Content-Type for Header Bar is "application/x-www-form-urlencoded"

Method:POST

SEND AUDIO

Descripton

Send audio to the server

Notice:The length of text size and audio size must be equal when user send to the server

REQUIREMENT

  • METHOD:POST
  • URL:/api/posts/sendaudio
  • Content-Type:form-data
  • Key:Value
    Text:"mytext"
    myFiles:"Myaudio"
